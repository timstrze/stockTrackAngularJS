line-chart
directive in module stockTrackAngularJsApp

Description

lineChart

Directive to create a line chart based off a Symbol's historical data.

Usage

as element:
<line-chart
       historical-data="{Array}">
</line-chart>

Parameters

ParamTypeDetails
historicalDataArray

An array of historical data

Methods

  • parseDate()

    Parses a date that is friendly to d3

  • render()

    Creates the line chart

Events

  • $on

    Watches the window and renders the line chart

  • $watch

    Watches the Symbol historicalData Array and calls the render Function.
