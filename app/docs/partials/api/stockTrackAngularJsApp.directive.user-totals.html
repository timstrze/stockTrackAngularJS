user-totals
directive in module stockTrackAngularJsApp

Description

userTotals

Directive to calculate the daily and total PNL and the User's total value

Usage

as attribute
<ANY user-totals>
   ...
</ANY>

Methods

  • calculateTotals()

    Calculates the daily and total PNL and the User's total value. Adds the Positions total PNL to the User's total PNL. Adds the Positions daily PNL to the User's daily PNL. Adds the Positions total value to the User's total value.

Events

  • $watch

    Watches the User Positions calls the calculateTotals Function.
