buy-button
directive in module stockTrackAngularJsApp

Description

buyButton

Directive to open the buy modal.

Usage

as attribute
<ANY buy-button
     user="{Object}"
     symbol="{Object}">
   ...
</ANY>

Parameters

ParamTypeDetails
userObject

User Object

symbolObject

Symbol Object that the User is going to buy

Methods

  • buy()

    Buys the selected symbol and quantity. Checks to see if the User has enough cash to make the trade. Updates the User's available cash.

  • openBuyModal()

    Opens the buy modal. Sets the default quantity to 1.
