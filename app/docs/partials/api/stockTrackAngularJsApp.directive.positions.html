positions
directive in module stockTrackAngularJsApp

Description

positions

Directive to display User Positions.

Usage

as attribute
<ANY positions="{Object}">
   ...
</ANY>

Parameters

ParamTypeDetails
positionsObject

User Positions

Methods

  • refreshSymbols()

    Pass through to the SymbolList.refreshSymbols method.

  • totalPositions()

    Calculates the PNL for the User Positions. If the buy did happen today use the Symbols ask value then use the Symbols previous close value.

Events

  • $watch

    Watches the User Positions and calls the totalPositions Function.
